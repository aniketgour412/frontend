# TravelVerse - College Project

TravelVerse is a beginner-friendly travel booking website created as a college project. This repository demonstrates core front-end web development structure using pure **HTML5** and **CSS3** without relying on JavaScript or external frameworks.

## Features

- **Multi-Page Architecture**: Contains independent HTML files for Home, Destinations, Packages, Forms, and Summary.
- **Pure CSS Layouts**: Makes heavy use of modern CSS techniques (Grid, Flexbox, Keyframes) to ensure a great responsive layout and dynamic UI (like hover transitions and CSS `:target` pseudo-selector routing).
- **Zero JavaScript**: Forms, UI interactions, and routing logic are handled natively via HTML `<form>` and `<a>` anchor attributes.
- **Easy Customization**: Built with explicitly clean semantic HTML and documented code blocks making it easy to explain in vivas and presentations.

## File Map

- `index.html` — Homepage featuring hero banner, popular destinations, and tour packages.
- `details.html` — Passenger details capture form and dynamic package state URL selection.
- `destination.html` / `goa.html` / `manali.html` / `dubai.html` — Modular destination-specific information layouts.
- `summary.html` — Invoice review layout.
- `payment.html` — Payment selection layout with a hidden CSS-only success receipt logic.

## Setup Instructions

Since this project relies purely on static front-end code, no server-side compilation is required.

1. Clone or download the repository to your local machine.
2. Locate the folder and simply double-click the `index.html` file.
3. The application will open directly inside your default web browser.
